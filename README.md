# HOI4 Console Scripts

yo, here be my HOI4 run scripts for quick console automation

buncha `.txt` files u yeet into ur HOI4 root folder (where `hoi4.exe` lives) n run with `run filename.txt` in console

saves u from typing 1000 commands manually like a pleb

## how 2 use

1. yeet `.txt` files into HOI4 root folder
2. open console (`~` or `ё`)
3. type: `run filename.txt`
